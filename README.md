# adapt-complete-button

##Installation

Download and include in the extensions folder of the adapt framework. 

Add the ids of the two elements you would like to switch between

{
  "_id": "adapt-switch",
  "_parentId": "parent",
  "_type": "component",
  "_component": "switchButton",
  "_classes": "",
  "_layout": "full",
  "value": "Switch",
  "switch": {
    "0" : "#first",
    "1" : "#second"
  },
  "displayTitle": ""
},
